int fin_de_partie_sdl ( Joueur **  j)

Vérifie si c'est vraiment la fin de la partie,modifie les scores et réalise les choix à faire.

Si le Joueur à une liste_vide, on le fait abandonner. Une fois que tous les Joueurs ont abandoné,mets à jour le score ,affiche les résultats dans la sdl et demande a l'utilisateur un choix:
-Recommencez une manche.
-Recommencez une partie.
-Quittez le programme.

Paramètres
jListe de tous les Joueurs pour vérifier s'ils ont tous abandonné.
Renvoie
Retourne le choix de l'utilisateur (ou 0 s'il reste un Joueur en jeu): 1 - Recommence une manche.
2 - Retourne au titre.
int initialisation_partie_sdl ( Joueur **  j)

Initialise une partie
Crée une liste de n Joueur [2-4].
Initialise le pseudo des joueurs Si la liste existe, on la supprime puis on en crée une autre.

Paramètres
jPointeur sur un Joueur pour créer la liste de Joueur.
Renvoie
Retourne 2 si le bouton retour a appuyer.
Retourne 3 si le joueur appuis sur la croix.
Retourne 4 si il a un problème reseau.
Retourne 0 si l'affectation a bien fonctionné.
