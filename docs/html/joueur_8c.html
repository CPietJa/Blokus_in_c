Description détaillée

Fichier de fonctions pour accesseurs et de modifications pour un Joueur.

Auteur
WIDMER Alexis
Version
1.0
Date
05/03/2019

Ce fichier permet d'avoir accès aux attributs d'un Joueur, de le créer ou de le supprimer.
